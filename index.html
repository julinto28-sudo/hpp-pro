Menjawab Keraguan Anda (FAQ)

Apakah data saya aman jika tidak memakai database MySQL/Firebase?

Sangat aman! HPP Master Pro menyimpan data menggunakan format JSON terenkripsi PHP pada file khusus di server hosting cPanel Anda. Server cPanel diprogram untuk melarang akses publik ke file tersebut (memicu error 403 Forbidden), sehingga database Anda tidak bisa dibaca, diunduh, atau disusupi dari luar.

Bagaimana cara kerja konversi satuan di aplikasi ini?

Sangat praktis! Saat memasukkan bahan baku, Anda dapat memilih satuan beli (misal: kg) dan satuan resep (misal: gram) serta mengisikan rasio konversi (1 kg = 1000 gr). Begitu Anda membuat resep, Anda tinggal memasukkan jumlah pemakaian dalam gram. Aplikasi akan langsung menghitung proporsi biaya secara otomatis.

Dapatkah aplikasi ini di-deploy di hosting murah?

Tentu saja! Aplikasi ini dirancang agar ringan tanpa perlu Node.js atau VPS di sisi server produksi. Anda bisa membeli shared hosting biasa senilai Rp15.000 - Rp25.000 per bulan, meng-upload file ZIP-nya, mengekstraknya di cPanel, dan aplikasi langsung siap melayani pelanggan Anda.
